## 🔩 Getting Started

```shell
$ git clone git@github.com:CodeAdvisor-ISTAD/code-advisor-ui.git
$ cd code-advisors.istad.co
$ npm install
$ npm run dev   # Open http://localhost:3000 with your browser to see the result.
```
